# AI-Powered QA Automation

This project is part of my QA Automation training where I’m learning how to build automated tests using Playwright and TypeScript.

## Tech Stack
- Playwright
- TypeScript
- Node.js

## About the Project
The goal of this project is to practice building automated test scenarios that simulate real user actions and help verify that the application works as expected.

I’m focusing on writing clean, maintainable tests and organizing them in a way that can scale as the project grows.

## What’s included
- Basic end-to-end test scenarios
- UI validation checks
- Structured test files

## Goal
To improve my automation skills and get comfortable with modern QA tools and practices.
