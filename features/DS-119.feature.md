Feature: DS-119 - Dashboard displaying the right components

  As an admin user, I want to see the correct dashboard so that I can
  navigate to the main areas of Didaxis Studio.

  # Happy paths

  Scenario: Navigate to the Dashboard
    Given I am logged in as admin
    When I navigate to the Dashboard page
    Then I see the Dashboard heading
    And I see the Programs dashboard card with subtitle "Manage academic programs"
    And I see the Calendar dashboard card with subtitle "Schedule & drag-drop"
    And I see the Validation dashboard card with subtitle "Check for conflicts"
    And I see the AI Assist dashboard card with subtitle "AI-powered editing"

  Scenario: Successfully navigate to Program Page from dashboard card
    Given I am logged in as admin
    And I am on the Dashboard page
    When I click the Programs dashboard card
    Then I navigate to the Programs page
    And I see the Programs heading

  Scenario: Successfully navigate to Calendar Page from dashboard card
    Given I am logged in as admin
    And I am on the Dashboard page
    When I click the Calendar dashboard card
    Then I navigate to the Calendar page
    And I see the Calendar heading

  Scenario: Successfully navigate to Validation Page from dashboard card
    Given I am logged in as admin
    And I am on the Dashboard page
    When I click the Validation dashboard card
    Then I navigate to the Validation page
    And I see the Validation heading

  Scenario: Successfully navigate to AI Assist Page from dashboard card
    Given I am logged in as admin
    And I am on the Dashboard page
    When I click the AI Assist dashboard card
    Then I navigate to the AI Assist page
    And I see the AI Assist heading

  # Negative

  Scenario: Dashboard does not show unrelated module cards as primary blocks
    Given I am logged in as admin
    And I am on the Dashboard page
    Then I do not see Scheduler as a primary dashboard block
    And I do not see Export as a primary dashboard block

  # Edge cases

  Scenario: Dashboard cards remain visible after returning from Programs
    Given I am logged in as admin
    And I am on the Dashboard page
    When I click the Programs dashboard card
    And I navigate back to the Dashboard via sidebar
    Then I see the Programs dashboard card with subtitle "Manage academic programs"
    And I see the Calendar dashboard card with subtitle "Schedule & drag-drop"

  Scenario: Dashboard welcome message is visible on load
    Given I am logged in as admin
    When I navigate to the Dashboard page
    Then I see the text "Welcome to Didaxis Studio"
    And I see the Connected status indicator
